Use the design system described in DESIGN.md through tailwindcss, prefer updating the theme to make tailwindcss comply with the design system.
Use shadcn with base ui and prefer modifying the shadcn component instead of local modifications when in need, shadcn components are your own.
When you need new components always check if shadcn has it and prefer using shadcn's than custom made ones, remember you can customize then to our needs and design system.
